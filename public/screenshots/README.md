# PWA Screenshots

Screenshots are used for the PWA install prompt on supported browsers.

## Required Screenshots

1. **home.png** (1280x720) - Wide screenshot for desktop
2. **mobile.png** (750x1334) - Narrow screenshot for mobile

## How to Create Screenshots

1. Run the app locally: `npm run dev`
2. Take screenshots of the homepage
3. Resize to the required dimensions
4. Save as PNG files in this folder

## Tips

- Use a clean browser without extensions
- Capture meaningful content (products, hero section)
- Ensure the design looks professional
- Test both light and dark modes
